# Dataset

This folder contains the dataset used for the house price prediction task.

## Description
- The dataset includes property-related features such as location, size, number of rooms, and other attributes.
- Data preprocessing includes handling missing values, feature encoding, and feature scaling.

## Target Variable
- **price**: The selling price of the house.

## Note
The dataset is not included in this repository.  
You can use any standard house price dataset (e.g., Kaggle house price datasets) to run the code.
